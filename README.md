# Thingsboard Javascript SDK

Thingsboard Javascript SDK


## Installation

```bash
npm install thingsboard-js-sdk
```

## Methods

- Constructor class (host, token)
- Connect (publicId)
- Auth (user/password)
- Disconnect
- Subscribe (websocket)
- getTenantDevices
- getKeys (timeseries & attributes)
- deleteEntityKeys (timeseries & attributes)
- getAttributesByScope
- getTimeseries
